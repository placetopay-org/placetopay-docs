# Solución a posibles problemas
A continuación se muestran algunos casos que se podrían presentar en el registro de usuarios, principalmente aquellas situaciones en las que se envía un correo electrónico al usuario y no puede ingresar al sistema.

1. Cuando el usuario presiona Completar Registro y no lo lleva al sistema, se recomienda copiar la URL que se muestra en la imagen, pegarla en el navegador para que sea redireccionado al sistema.

<img src="/Selfdeclarations/complete_registration_button.png"/>

2. Si el usuario al seleccionar el botón Ir a mi perfil tiene problemas de conexión, podrá copiar la URL que se muestra en la imagen en su navegador para acceder.

<img src="/Selfdeclarations/go_to_my_profile.png"/>

3. En caso que un usuario olvide su contraseña la podrá restablecer dando clic en ¿Olvidaste tu contraseña? en la pantalla de inicio de sesión.

<img src="/Selfdeclarations/reset_password.png"/>

Se ingresa el correo electrónico registrado por el usuario, se da clic en Restablecer y el sistema enviará un correo para restablecer la contraseña.

4. Si al completar el registro de un declarante se genera el siguiente mensaje:
> Lo sentimos, el usuario con documento 12345678, no se encuentra asociado a la empresa con documento 1234567890

Significa que el usuario administrador debe asociar previamente al usuario con la empresa correspondiente. Hasta que no se realice esta asociación, no será posible completar el proceso de registro. [Ver documentación](/selfdeclarations/admin-user/actions#empresas).
<img src="/Selfdeclarations/error_message_user_not_related.png"/>

5. Si al completar el registro de un declarante se genera el siguiente mensaje:
> Lo sentimos, el usuario con documento 1234567, no tiene una renta asociada en la empresa con documento 123456789"

Significa que un administrador debe asociar al menos una renta al usuario autorizado dentro de la empresa correspondiente. De lo contrario, no se podrá completar el registro. [Ver documentación](/selfdeclarations/admin-user/actions#empresas).

<img src="/Selfdeclarations/error_register_user_declaration.png"/>

6. Cuando debes realizar un pago y no se ha autorizado la declaración para realizar la firma electrónica porque no es visible, se debe comunicar con el administrador del sistema para que habilite la opción.

<img src="/Selfdeclarations/enable_payment_option.png"/>
