# Empresas - Creación/Actualización

Permite crear/actualizar una empresa en el sistema de Autodeclaraciones. Las empresas son todas aquellas entidades que deben declarar los impuestos.

La empresa se puede inactivar usando el parametro status

Permisos requeridos sobre el usuario API: manage-companies.api
