# Entrada y salida del software

## Ingreso
El sistema permite a los funcionarios del ente territorial acceder al aplicativo con perfiles predeterminados con el fin de que cada uno tenga acceso a la información necesaria para desempeñar sus funciones.

Una vez dentro del software, para iniciar sesión se da clic en **Ingresar**
<img src="/Selfdeclarations/Declarant_user/entry_and_exit/login_user_btn.png"/>

Se abrirá el formulario de inicio de sesión, en el cual se debe seleccionar el tipo de documento, ingresar el documento, la contraseña, y por último presionar **Ingresar**.
<img src="/Selfdeclarations/Declarant_user/entry_and_exit/login_user_form.png"/>

## Salida
Para salir del sistema se podrá hacer de varias formas.

* La forma más segura es seleccionar el icono que se encuentra en la parte superior derecha de la ventana, donde se despliega una lista como se muestra a continuación y allí seleccionar la opción, **Cerrar sesión**.
<img src="/Selfdeclarations/Declarant_user/entry_and_exit/logout_user_btn.png"/>

* Se podrá salir del sistema cerrando la pestaña del navegador o cerrando directamente el navegador, sin embargo la sesión no se cerrará automáticamente, por lo que al ingresar a la URL de nuevo, es posible que entre con la sesión iniciada generando brechas de seguridad.
