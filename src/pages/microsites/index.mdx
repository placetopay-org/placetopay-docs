# Introducción a Micrositios

Los micrositios son una solución ideal para vender y recaudar en línea sin la necesidad de tener un sitio web completo. Estos sitios permiten una personalización detallada con tu marca y facilitan la interacción directa con tus clientes.

Con nuestros micrositios, podrás ofrecer a tus clientes una plataforma sencilla y eficaz para realizar pagos de impuestos, servicios, seguros, viajes, suscripciones, membresías, donaciones, entre otros.

Además, proporcionamos una API robusta para la creación de micrositios, ya sean abiertos o cerrados. Esto te permite ajustar las configuraciones de pago según las necesidades específicas de tu negocio.

Explora nuestra [API de Micrositios](/microsites/api/reference/microsites) para conocer más detalles y empezar a integrar esta poderosa herramienta en tu estrategia de ventas y recaudación en línea.
