# Reglas de Recurrencia

El objeto `recurring` es el componente principal de la integración. Define la frecuencia, duración y comportamiento del contrato de pago.

## Periodicidad e Intervalo

La frecuencia de cobro se determina combinando `periodicity` (unidad de tiempo) e `interval` (multiplicador).

| Unidad (`periodicity`) | Descripción |
| :--- | :--- |
| **`D`** | Días |
| **`W`** | Semanas |
| **`M`** | Meses |
| **`Y`** | Años |

### Ejemplos de Configuración

| Frecuencia Deseada | Configuración JSON |
| :--- | :--- |
| **Mensual** | `"periodicity": "M", "interval": 1` |
| **Trimestral** | `"periodicity": "M", "interval": 3` |
| **Quincenal** | `"periodicity": "W", "interval": 2` |
| **Anual** | `"periodicity": "Y", "interval": 1` |

## Control del Ciclo de Vida

Adicionalmente, puedes controlar cuándo inicia y cuándo termina el AutoPago:
