# Estados de un Autopago



| Estado   | Descripción                                                                          |
|----------|--------------------------------------------------------------------------------------|
| APPROVED | Indica un autopago aprobado                                                          |
| REJECTED | Indica un autopago rechazado y puede estar relacionado con diferentes reason code's. |
| FAILED   | Indica un autopago fallido y puede estar relacionado con diferentes reason code's.   |



---

# Reason Code's que puede tener un Autopago



| Reason Code | Descripción                      |
|-------------|----------------------------------|
| 00          | Indica un autopago aprobado      |
| ?C          | Indica un autopago cancelado     |
| AP          | Indica un autopago completado    |


---

# Cambio del estado de un Autopago según escenario

| Escenario                                                                                                                              | Estado   | Reason |
|----------------------------------------------------------------------------------------------------------------------------------------|----------|--------|
| Creación de autopago                                                                                                                   | APPROVED | 00     |
| Cancelación de un autopago                                                                                                             | REJECTED | ?C     |
| Autopago completado. La ultima transacción del autopago es aprobada.                                                                   | REJECTED | AP     |
| Se alcanza el límite de pagos atrasados permitidos.                                                                                    | FAILED   | ?C     |
| Se alcanza el límite de reintentos por indisponibilidad del servicio externo en la consulta de balance durante el registro de un pago. | FAILED   | ?C     |
| Se alcanza el límite de reintentos por fallo técnico bien sea durante el registro o procesamiento de la cuota de un autopago.          | FAILED   | ?C     |
