# AutoPay (BETA)

**AutoPay Placetopay** es un servicio de pagos automáticos que permite a los comercios habilitar cobros recurrentes y programados sin intervención manual. La solución gestiona la seguridad (tokenización PCI) y la orquestación de los cobros (scheduling), permitiendo modelos de suscripción fija o facturación variable.

## Guía de Integración

La documentación está dividida en dos áreas según tu rol en la interacción:

### 1. API de Gateway (Tu servidor -> Placetopay)
Utiliza estos servicios para crear suscripciones, gestionar usuarios y consultar estados.

* [**Cómo funciona:**](/autopay/how-it-works) Entiende la arquitectura Proxy y el ciclo de vida.
* [**Reglas de Recurrencia:**](/autopay/recurring-setup) Configura correctamente frecuencias (Días, Meses) y reglas de vencimiento.
* [**Crear AutoPago:**](/autopay/api/session#create-a-session) Endpoint principal para iniciar suscripciones.
* [**Gestión:**](/autopay/api/session/#query-a-session) Consulta, cancela o revisa el historial de tus autopagos.

### 2. Contratos del Cliente (Placetopay -> Tu servidor)
Debes implementar estos endpoints en **tu servidor** si vas a utilizar cobros de monto variable (Ej. Facturación de servicios públicos).

* [**Autenticación de Contratos:**](/autopay/contract-authentication) Seguridad para validar nuestras peticiones.
* [**Balance:**](/autopay/contracts/balance) Permite a AutoPay consultar cuánto debe el usuario antes de cobrar.
* [**Settlement:**](/autopay/contracts/settlement) Confirma la aplicación del pago en tu ERP.
* [**Webhooks:**](/autopay/contracts/webhook) Recibe notificaciones de cobros, errores y cambios de estado.
