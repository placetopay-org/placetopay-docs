# AutoPay (BETA)

**AutoPay Placetopay** es un servicio de pagos automáticos que permite a los comercios habilitar cobros recurrentes y programados sin intervención manual.

## Guía de Integración

La documentación está dividida en dos grandes áreas según tu rol en la integración:

### 1. API de Gateway (Tu servidor -> Placetopay)
Utiliza estos servicios para crear suscripciones, gestionar usuarios y consultar estados.

* [**Cómo funciona:**](/autopay/how-it-works) Entiende el flujo y la arquitectura.
* [**Crear AutoPago:**](/autopay/session) Aprende a crear una sesión de suscripción.
* [**Reglas de Recurrencia:**](/autopay/recurring-setup) Configura correctamente las fechas y frecuencias.
* [**Gestión:**](/autopay/search) Consulta, cancela o revisa el historial de tus autopagos.

### 2. Contratos del Cliente (Placetopay -> Tu servidor)
Implementa estos endpoints si vas a realizar cobros de montos variables (Facturación).

* [**Autenticación de Contratos:**](/autopay/contract-authentication) Seguridad para recibir nuestras peticiones.
* [**Balance:**](/autopay/contracts/balance) Permite a AutoPay consultar cuánto debe el usuario.
* [**Settlement:**](/autopay/contracts/settlement) Confirma la aplicación del pago en tu sistema.
* [**Webhooks:**](/autopay/contracts/webhook) Recibe notificaciones de cobros y cambios de estado.
