## Crear una lista de preferidos {{ id: 'post-core-site-prefer-list-create', tag: 'POST', label: '/core/site/prefer-list/create' }}

<Row>
    <Col>
        Este servicio permite crear una lista de preferidos y asociarla a un sitio existente. Si la respuesta es exitosa, retorna el identificador de la lista.

> ### Nota 
>
> *Para crear correctamente la lista de preferidos es necesario ingresar junto a los campos requeridos, por lo menos uno de los siguientes campos: card.Number, ipAddresses o person.document.*

> *En el objeto Person, es requerido enviar el campo email. Los campos name, surname, company, address y mobile no son necesarios al enviar la solicitud.*
    
        <ApiReader
            path="/core/site/prefer-list/create"
            method="post"
            api={props.refs}
        />
    </Col>
        <Col sticky>
        <CodeGroup title="Solicitud" tag="POST" label="/core/site/prefer-list/create">
```bash {{title: 'Creación de una lista de preferidos'}}
curl -X "POST" https:\test.placetopay.com\rest\core\site\prefer-list\create \
-H "Content-Type: application/json" \
-d '{
  "auth": {
    "login": "613822f45568191a8cdf4ad2299f6d23",
```

## Consulta de lista de preferidos {{ id: 'post-core-site-prefer-list-search', tag: 'POST', label: '/core/site/prefer-list/search' }}

<Row>
    <Col>
        Este servicio permite consultar listas de preferidos existente filtrando por medio de los parámetros: siteId, id, cardDisplay, email, document e ipAddresses.

Si requiere el uso de paginación personalizada, debe enviar el parámetro 'pagination' junto a los filtros de la busqueda.

> ### Nota 
>
> *No es requerido ingresar todos los parámetros para el filtro.*
    
        <ApiReader
            path="/core/site/prefer-list/search"
