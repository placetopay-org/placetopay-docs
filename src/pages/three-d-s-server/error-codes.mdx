# Códigos de Error

Los siguientes son los códigos de error establecidos por 3DSS Placetopay, con su respectiva descripción y código HTTP asociado.

<Tabs>
  <Tabs.Item title="Códigos de error API de sesión">
 Código  | Código HTTP  | Descripción                                                                              |
 --------|--------------|------------------------------------------------------------------------------------------|
 1002    | 422          |Argumentos no válidos para iniciar una petición.                                          |
 1003    | 404          |No se encontró un comercio asociado.                                                      |
 1004    | 404          |No se encontró una suscripción asociada.                                                  |
 1005    | 403          |La franquicia asociada no se encuentra habilitada.                                        |
 1006    | 403          |El adquiriente asociado no se encuentra habilitado.                                       |
 1007    | 403          |La suscripción asociada no está completamente configurada.                                |
 1011    | 403          |El comercio no se encuentra habilitado.                                                   |
 1012    | 403          |No se encontró un servidor de directorio para el protocolo seleccionado.                  |
  </Tabs.Item>
  <Tabs.Item title="Códigos de error API de comercios y sucursales">
 Código  | Código HTTP  | Descripción                                                                              |
 --------|--------------|------------------------------------------------------------------------------------------|
 1001    | 422          |Argumentos requeridos no presentes en la petición.                                        |
 1002    | 422          |Argumentos no válidos para iniciar una petición.                                          |
 1003    | 404          |Nombre de comercio no disponible.                                                         |
 1004    | 404          |Nombre de sucursal no disponible.                                                         |
 1005    | 403          |La franquicia asociada no se encuentra habilitada (suscripción no posible).               |
 1006    | 403          |El adquiriente asociado no se encuentra habilitado (suscripción no posible).              |
 1007    | 403          |La suscripción asociada no está completamente configurada.                                |
 1008    | 422          |Directory server inhabilitado.                                                            |
 1009    | 404          |El bin del adquiriente asociado no existe (suscripción no posible).                       |
 1010    | 404          |Comercio no encontrado.                                                                   |
  </Tabs.Item>
</Tabs>
