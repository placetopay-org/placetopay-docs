# Payment Button - AIO

This service allows you to create a payment button by sending all the required information in a single request.

<Note>
    **IMPORTANT:**

    Only the users included in the request will be able to interact with the declaration during its authorization process.
</Note>

The permissions and responses provided by this service are identical to those provided by the Companies, Users, and General Payment Button services.

That is, if there is an error during the creation/update of the company, for example, the same response codes associated with the company creation service will be sent for users and the payment button.

The API behaves differently depending on the type of return, so you should be aware of some special fields:

<code>additionalData.valueStrings</code> and <code>additionalData.valueArrays</code>. These fields contain each additional form data item as objects with the keys <code>keyword</code> and <code>value</code>. The possible values for <code>keywords</code> and <code>values</code>, depending on the type of return, are:
