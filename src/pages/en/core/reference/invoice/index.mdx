## Block or release invoices. {{ id: 'post-core-invoice-hold', tag: 'POST', label: '/core/invoice/hold' }}

<Row>
    <Col>
        This service allows you to block or release one or more invoices.

        When blocked, the invoice will show a payment status of “in process” in the PlacetoPay admin panel. It is not possible to edit or delete a blocked invoice.
        <ApiReader
            path="/core/invoice/hold"
            method="post"
            api={props.refs}
        />
    </Col>
    <Col sticky>
        <CodeGroup title="Request" tag="POST" label="/core/invoice/hold">

            curl -X "POST" https:\test.placetopay.com/rest/core/invoice/hold \
            -H "Content-Type: application/json" \
            -d '{
                "auth": {
                    "login": "aabbccdd1234567890aabbccdd123456",
