## Ads search {{ id: 'post-core-ads-search', tag: 'POST', label: '/core/ads/search' }}

<Row>
    <Col>
      This service allows you to consult the advertisements available for the site.

      **Only returns active ads** (where end_date is greater than or equal to the current date). Results are sorted by start_date in ascending order.

      It is possible to use filtering through the parameters: "type".

      > ### Note
      >
      > *It is not required to enter all the parameters for the filter.*

      If you require the use of custom pagination, you must send the 'pagination' parameter along with the search filters.
