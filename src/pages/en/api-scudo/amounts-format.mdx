# Amounts Format

To send amount values, they must be sent as integers and not decimals, converting the decimal value to the smallest currency unit format. [ISO 4217](https://www.iso.org/iso-4217-currency-codes.html)

## Formula

**Currency:** COP <br/>
**Minor currency unit:** 2 <br/>
**Decimal amount:** 23,887.90 <br/>
**Formula:** Decimal amount * (10^(Minor currency unit)) <br/>
**Total:** 23,887.90 *(10^2) = 2388790 <br/>

## Examples

| Currency | Minor unit | Decimal   | Integer  |
| :------- | :--------- | :-------- | :------- |
| COP      | 2          | 23,887.90 | 2388790  |
| USD      | 2          | 458.48    | 45848    |
