# Formato de montos

Para enviar valores montos, se deben enviar en formato entero y no decimal, cambiando el valor decimal al formato con la unidad menor.
[ISO 4217](https://www.iso.org/iso-4217-currency-codes.html)

## Fórmula

**Moneda:** COP <br/>
**Unidad menor de moneda:** 2 <br/>
**Monto decimal:** 23,887.90 <br/>
**Fórmula:** Monto decimal * (10^(Unidad menor de moneda)) <br/>
**Total:** 23,887.90 *(10^2) = 2388790 <br/>

## Ejemplos

| Moneda | Unidad menor | Decimal   | Entero  |
| :----- | :----------- | :-------- | :------ |
| COP    | 2            | 23,887.90 | 2388790 |
| USD    | 2            | 458.48    | 45848   |
