# Signature

To create a signature in the HTTP headers, you need the following data:

**Secret Key:** Secret key provided by Evertec Placetopay. <br/>
**Seed:** Timestamp. <br/>
**URL:** Webhook URL. <br/>

## Example calculation in PHP
```php
    $url = 'https://client-url/notification';
    $seed = now()->timestamp; // 1648159035
    $secretKey = 'thisIsAnExampleOfSecretKey';

    $signature = hash_hmac('SHA256', $url . $seed, $secretKey);
```
