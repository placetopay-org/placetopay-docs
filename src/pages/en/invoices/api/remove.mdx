# Delete payment order

This endpoint allows you to **permanently delete** a payment order.

<Note type='warning'>
- Payment orders that have already been processed or locked are not deleted.
- This operation cannot be undone.
</Note>

The operation requires the **unique ID of the payment order**, generated by the system at the time of its creation.
The response returns the complete information of the payment order **exactly as it was registered before being deleted**.
