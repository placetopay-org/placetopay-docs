# Token

The key generated by a subscription process is identified as a token, allowing the generation of charges without user interaction. {{ className: 'lead' }}

## Invalidate token {{ id: 'invalidate-token', tag: 'POST', label: '/api/instrument/invalidate' }}

<Row>
  <Col>
    Allows you to invalidate an existing site token. This will make the token or subtoken no longer usable.

    <ApiReader
      path="/api/instrument/invalidate"
      method="post"
      api={props.refs}
    />
  </Col>
  <Col sticky>
    <CodeGroup title="Request" tag="POST" label="/api/instrument/invalidate">
```bash {{ title: 'Using Token' }}
curl -X "POST" https://checkout-test.placetopay.com/api/instrument/invalidate \
  -H "Content-Type: application/json" \
  -d '{
    "locale": "en_US",
    "auth": {
      "login": "c51ce410c124a10e0db5e4b97fc2af39",
      "tranKey": "VQOcRcVH2DfL6Y4B4SaK6yhoH/VOUveZ3xT16OQnvxE=",
      "nonce": "NjE0OWVkODgwYjNhNw==",
      "seed": "2021-09-21T09:34:48-05:00"
    },
    "instrument": {
      "token": {
        "token": "a3bfc8e2afb9ac5583922eccd6d2061c1b0592b099f04e352a894f37ae51cf1a"
      }
    }
  }'
```
```bash {{ title: 'Using Subtoken' }}
curl -X "POST" https://checkout-test.placetopay.com/api/instrument/invalidate \
  -H "Content-Type: application/json" \
  -d '{
    "locale": "en_US",
    "auth": {
      "login": "c51ce410c124a10e0db5e4b97fc2af39",
      "tranKey": "VQOcRcVH2DfL6Y4B4SaK6yhoH/VOUveZ3xT16OQnvxE=",
      "nonce": "NjE0OWVkODgwYjNhNw==",
      "seed": "2021-09-21T09:34:48-05:00"
    },
    "instrument": {
      "token": {
        "subtoken": "8740257204881112"
      }
    }
  }'
```
