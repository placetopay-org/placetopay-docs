## Invalidate token {{ id: 'invalidate-token', tag: 'POST', label: '/api/instrument/invalidate' }}

<Row>
  <Col>
    Allows you to invalidate an existing site token. This will make the token or subtoken no longer usable.

    <ApiReader
      path="/api/instrument/invalidate"
      method="post"
      api={props.refs}
    />
  </Col>
  <Col sticky>
    <CodeGroup title="Request" tag="POST" label="/api/instrument/invalidate">
      ```bash {{ title: 'cURL' }}
      curl -X "POST" https://checkout-test.placetopay.com/api/instrument/invalidate \
        -H 'Content-Type: application/json' \
        -d '{
          "auth": {
            "auth": {
              "login": "c51ce410c124a10e0db5e4b97fc2af39",
      ```
