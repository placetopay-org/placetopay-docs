## Cancel an AutoPay {{ id: 'cancel-autopay', tag: 'POST', label: '/autopay/cancel'}}
Its purpose is to cancel/disable an autopay so that previously scheduled automatic/recurring charges are not executed.

<Note type="warning"> Once the autopay is canceled, it cannot be reactivated. If you need an autopay again, you must create a new one. </Note>
