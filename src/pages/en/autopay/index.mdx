# PlacetoPay AutoPay (BETA)

**PlacetoPay AutoPay** is an automated payment service that allows merchants to enable recurring or scheduled charges using a payment method previously authorized by the user. {{ className: 'lead' }}

The solution includes a guided enrollment experience for the end user, API services for merchants to create and manage AutoPay sessions, and charge execution processes based on the defined configuration.

<Note>
  **Important:** This feature must be explicitly enabled on your site.
</Note>
