# Get authentication information

This service allows you to query the status of a transaction that has gone through the cardholder authentication process with the 3DS Server.

## Service Data
|          |                                          |
|----------|------------------------------------------|
| Endpoint | **/api/v2x/transactions/`{transactionID}`** |
| Method   | GET                                      |

**Note:** `{transactionID}` should be taken from the transactionID data in [Create Session](/en/three-d-s-server/api/sessions)

---

## Possible errors

| Code | Cause                                       |
|------|---------------------------------------------|
| 403  | The token is not authorized.                | 
| 404  | Transaction does not exist                  | 
| 500  | The server has encountered an internal error. |
