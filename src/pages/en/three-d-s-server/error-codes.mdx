# Error Codes

The following are the error codes established by 3DSS Placetopay, along with their descriptions and associated HTTP status codes.

<Tabs>
  <Tabs.Item title="Session API Error Codes">
 Code   | HTTP Code     | Description                                                                               |
 -------|---------------|-------------------------------------------------------------------------------------------|
 1002   | 422           |Invalid arguments provided to initiate a request.                                          |
 1003   | 404           |No associated merchant found.                                                              |
 1004   | 404           |No associated subscription found.                                                          |
 1005   | 403           |The associated card brand is not enabled.                                                  |
 1006   | 403           |The associated acquirer is not enabled.                                                    |
 1007   | 403           |The associated subscription is not fully configured.                                       |
 1011   | 403           |The merchant is not enabled.                                                               |
 1012   | 403           |No directory server found for the selected protocol.                                       |
  </Tabs.Item>

  <Tabs.Item title="Merchant and Branch API Error Codes">
 Code   | HTTP Code     | Description                                                                               |
 -------|---------------|-------------------------------------------------------------------------------------------|
 1001   | 422           |Required arguments missing in the request.                                                 |
 1002   | 422           |Invalid arguments provided to initiate a request.                                          |
 1003   | 404           |Merchant name not available.                                                               |
 1004   | 404           |Branch name not available.                                                                 |
 1005   | 403           |The associated card brand is not enabled (subscription not possible).                      |
 1006   | 403           |The associated acquirer is not enabled (subscription not possible).                        |
 1007   | 403           |The associated subscription is not fully configured.                                       |
 1008   | 422           |Directory server is disabled.                                                              |
 1009   | 404           |Acquirer BIN does not exist (subscription not possible).                                   |
 1010   | 404           |Merchant not found.                                                                        |
  </Tabs.Item>
</Tabs>
