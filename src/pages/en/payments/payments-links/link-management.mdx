## Related payments {{ id: 'relationship-payments', tag: '', label: '' }}

At the end of the whole process, in the view corresponding to the payment link created, a detailed record of the transaction will be automatically generated in the Related Payments section. In this section, you will be able to view all the associated payments that have been processed through that specific payment link. {{ className: 'lead' }}

<Note>Please note that the related payment shows only basic information. To view full details of payments received, it is recommended to view them from the administrative panel.</Note>

In case you do not see any expected record, it is important to verify that the link has been used correctly and that the payment flow has been completed without interruptions. {{ className: 'lead' }}

## Search for payment links {{ id: 'search', tag: '', label: '' }}

To search in the Payment Links history, go to the left-side menu of the platform and select the **Payment Links** option. Then click on **Consult** to view the history based on the preset filter, which corresponds to the **creation date**.{{ className: 'lead' }}
If you want to perform a more specific search, you can remove the default filter and use the **Add filter** option. This will allow you to set custom criteria based on the available options, displaying only the information that matches the selected parameters. This way, you will have more precise and efficient access to the relevant data.{{ className: 'lead' }}
