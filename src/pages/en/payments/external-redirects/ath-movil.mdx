ATH Móvil allows the payer to approve the transaction from their phone, while the merchant handles status confirmation according to the integration model.

## ATH Móvil Benefits

- **Instant transfers**: Send and receive money in real time using a mobile phone number.
- **Broad financial participation**: Works with participating banks and credit unions in the ATH Móvil ecosystem.
- **Security and speed for daily transactions**: Delivers a secure experience for everyday payments and transfers.

## Use Cases

- **Merchants in Puerto Rico**: Ideal for businesses that want to enable ATH Móvil as a local payment option.
- **Second-device payment approval**: The buyer confirms the transaction on their phone while keeping checkout open.

## Payment Method Behavior

ATH Móvil is categorized as a redirection payment method in the integration model, where approval is completed on the payer's phone and the session is updated with the transaction status.

![ATH Móvil payment flow](/en/payments/ath-movil/pm-ath-movil-flow.png)

### Webcheckout Flow

1. The user selects **ATH Móvil** as the payment method.
2. The user completes their payment information and confirms the payment.
3. The payment remains pending while the user approves from their phone.
4. Webcheckout periodically checks the transaction status.
5. When the status changes, the session updates and shows the final result.

- **Integrations through**: [Webcheckout](/en/checkout)
  The user approves the payment from their phone and the Checkout session updates with the final transaction outcome.

- **Integrations through**: [Gateway](/en/gateway)
  Integration is asynchronous. After creating/processing the transaction, the integrator must query status via API until the final result is obtained.

  To get the final transaction status, send a request to [Query transaction by internal reference](/en/gateway/api/reference/transaction#query-request).

## Restrictions and Considerations

- **Asynchronous Gateway flow**: The merchant must implement API status polling until a final result is available.
- **Pending state during approval**: The transaction remains pending while the user confirms the payment on their phone.
- **Regional availability**: ATH Móvil applies to operations in Puerto Rico.
