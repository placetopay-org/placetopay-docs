# Custom Microsite
<br/>
This service allows the consultation and management of payment invoices by connecting to the merchant's system. 
In this section, it is explained how to search for invoices and settle a payment on the platform to the merchant's system.

<Note>To interact with this service, it is necessary to use a programming language compatible with REST APIs, which allows efficient communication between systems.</Note>

The integration method supports the following key operations:

- Invoice lookup by reference, document number or alternate reference.
- Recording of payments with automatic update of invoice status once payment is processed.
