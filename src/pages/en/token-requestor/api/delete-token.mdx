# EnrollCard

Its function is to eliminate the brand's token so that the consumer can
no longer use the token to create cryptograms and use them in subsequent
payments to different providers, nor can they query data about this token.
Before consuming this API, a merchant registered with the brand and a previously
assigned token must exist on the platform.

---

<Note type="warning">
    The currently documented reason codes are not final and may be updated in future releases.
</Note>
