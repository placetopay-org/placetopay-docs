# GetTokenStatus

This endpoint allows you to query the current status of a brand token.
Its function is to return updated token data, avoiding failed attempts at
cryptogram generation and optimizing payment processing.

Before consuming this API, a merchant registered with the brand and
a previously specified token must exist on the platform.
