# Tokenización y Cobros masivos

## Tokenización

La tokenización es un servicio que permite convertir la información plana de las tarjetas en tokens que la representan, ofreciendo una forma mucho más segura de almacenarla y gestionarla.

## Cobros masivos

Este servicio permite procesar transacciones de forma masiva mediante archivos de entrada y salida, evitando la generación de múltiples consumos individuales al API.
