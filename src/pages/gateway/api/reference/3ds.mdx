## 3DS Query {{ id: 'mpi-query-request', tag: 'POST', label: '/gateway/mpi/query' }}

Una vez que se inicia una sesión de autenticación y luego de generarse las validaciones necesarias se puede consultar el estado de la misma para determinar si el titular de la tarjeta ha sido autenticado correctamente.

> ### Nota Importante
>
> *Al consumir este servicio **NO** se está autenticando la transacción por defecto, se debe tomar de la respuesta el valor de `data` y enviarlo en el `instrument.threeDS`*
