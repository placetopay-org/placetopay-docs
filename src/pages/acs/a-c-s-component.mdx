# Acerca del componente ACS

El componente ACS o Access Control Server, opera en el Dominio del Emisor del protocolo 3-D Secure. 

Este componente verifica si la autenticación está disponible para un número de tarjeta y tipo de dispositivo, procediendo a autenticar al titular de la tarjeta en específico. Además, este componente contiene las reglas para validar las autenticaciones.

### Este componente se encarga de:

- Verificar si un número de tarjeta es elegible para la autenticación 3-D Secure.

- Verificar si un tipo de dispositivo de consumidor es elegible para la autenticación.

- Autenticar al titular de la tarjeta para una transacción específica.

- Confirmar la información de una cuenta para transacciones de tipo 3RI (no pago).
