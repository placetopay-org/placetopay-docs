# Buscar orden de pago

Este endpoint permite consultar la información completa de las ordenes de pago, utilizando el **documento del deudor**, la **referencia** o la **referencia alterna**.
La respuesta devuelve no solo la orden de pago solicitada, sino también todas las ordenes asociadas al documento vinculado.
