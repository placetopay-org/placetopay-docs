# API de administración y gestión de tokens

---
## Información General

Esta API proporciona un conjunto de métodos que permiten la tokenización de PAN. Dado un PAN, puedes devolver un token y usarlo para pagos de comercio electrónico y móvil. Estas APIs se conocen como gestión de tokens y también proporcionan un conjunto de notificaciones a las que puedes suscribirte para cualquier cambio de estado del token.

Típicamente, un cliente de Token requestor inscribe a un comerciante, inscribe un PAN y proporciona un token dado un dato PAN. El cliente puede luego obtener datos de pago y gestionar tokens (por ejemplo, suspender, reanudar o eliminarlos).

Los clientes de Token requestor deben manejar notificaciones de eventos relacionados con el estado del token (creado, actualizado, clave cambiada), metadatos del PAN y transacciones de pago. Específicamente, el cliente debe llamar a la API de estado del token cuando recibe una notificación de estado del token.

- **Versión**: 1.0.0
