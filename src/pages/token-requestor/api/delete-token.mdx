# EnrollCard

Su función es eliminar el token de la marca para que el consumidor ya no pueda utilizar el
token para crear criptogramas y usarlos en pagos posteriores en los diferentes proveedores,
ni consultar datos sobre este toke
Antes de consumir esta API debe existir en la plataforma un comercio registrado en la
marca y un token anteriormente dado.
