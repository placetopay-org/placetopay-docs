# Tipos de documento

La API de Placetopay soporta diferentes tipos de documento dependiendo del país.

Para definir o usar los atributos `payer`, `buyer`, `shipping` y otros, debes anexar los atributos `document` y `documentType`. Estos datos son validados por la API para asegurar el formato de la información.
