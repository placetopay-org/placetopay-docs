### PDF

•	Transacciones exitosas, fallidas y las devoluciones de ACH.

- **Daily Summary Report**
    - Report Date
    - Report Time
    - Batch Number
    - Merchant Name
    - Total (Each Payment Methods*) Payments
    - Total (Each Payment Methods*) Amount
    - Total ACH Payments
    - Total ACH Payment Amount
    - Total ACH Return Payments
    - Total ACH Return Payment Amount
    - Total Cash
    - Total Cash Amount
    - Total Cards Payments
    - Total Cards Amount
    - **Nota:** Los medios de pago: AMEX, VISA, MASTERCARD, DISCOVER, ATH, ATH MOVIL, ACH, EBT (Todos estan actualmente habilitados)

### Successful Payments
    - Transaction Count #
    - Date
    - Time
    - CustomerAccountNumber
    - Authorization #
    - Last 4 Digits
    - Payment Type
    - Payment Description
    - Response
    - Amount ($)
    - Batch Number

### Unsuccessful Payments
    - Transaction Count #
    - Date
    - Time
    - CustomerAccountNumber
    - Authorization #
    - Last 4 Digits
    - Payment Type
    - Payment Description
    - Response
    - Amount ($)
    - Batch Number

### ACH Returns
    - Transaction Count #
    - Date
    - Time
    - CustomerAccountNumber
    - Reason Code (NACHA code)
    - Reason Code (NACHA code) Description
    - Payment Description
    - Authorization #
    - Last 4 Digits
    - Payment Type
    - Amount ($)
    - Batch Number
