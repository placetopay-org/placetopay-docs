# Creación de usuario

Esta sección guía paso a paso a los nuevos usuarios en el proceso de registro y acceso seguro a la plataforma Dash de Evertec PlacetoPay. {{ className: 'lead' }}

## Paso 1 Aceptación de invitación

La creación de un usuario inicia a partir de una invitación enviada por correo electrónico. Para completar el registro, el usuario debe dar clic al botón Aceptar invitación incluido en el correo.

<CenteredContainer>
   ![aceptarinvitacion](/payments/newpanel/migration/newuser/invitacion.webp)
</CenteredContainer>

Al dar clic en el botón **Aceptar invitación** el sistema lo redirige al formulario de creación de cuenta, donde se van a solicitar los campos: 

- **Correo electrónico**: Permite asignar un correo el cual quedara vinculado al usuario. 

- **Nombre**: Permite asignar nombre al usuario a crear. 

- **Contraseña**: Permite configurar una contraseña, la cual permitirá el acceso a nuestro Dash. 

- **Confirmación de contraseña**: Permite confirmar la contraseña ingresada en el campo anterior. 

![createuser](/payments/newpanel/migration/newuser/confirmvalidationes.webp)

El sistema lo debe redirigir a la pantalla donde al correo electronico le llegara un codigo el cual debe de copiar en este apartado.

![confirmacionotp](/payments/newpanel/migration/newuser/codeotpes.webp)

Al completar la información del login de clic en la validación reCaptcha, de ser necesario cumpla con el reto para continuar, seguido de clic en el botón **Validar Código**.  

## Paso 2 Configuración doble factor de autenticación (2FA)

A continuación se muestra la información correspondiente al doble factor de autenticación, el cual se debe cumplir para poder acceder al Dash, para esto se le brindará dos opciones:
- Ingresar por medio de un código por correo electronico 
- Por medio de una App de autenticación.  

![configuracion2fa](/payments/newpanel/migration/newuser/configuracion2fa.webp)

Si selecciona la opción de **Código a su correo**, le llegara un mensaje con al correo electronico y luego de verificar el código podrá generar el ingreso al Panel. [Inicio de sesión](/payments/newpanel/users/createuser/#paso-3-inicio-de-sesion-al-dash).

![2faemail](/payments/newpanel/migration/newuser/2faemail.webp)

Si selecciona la opción de **App de Autenticación** verá las siguientes opciones:

- Descargue una de las aplicaciones sugeridas (Validar la siguiente imagen) 

![2faapp](/payments/newpanel/migration/newuser/2faapp.webp)

- Escanee el codigo de Qr con la aplicación descargada. 

![confirmacion2faapp](/payments/newpanel/migration/newuser/confirmacion2faapp.webp)

- Finalmente, ingrese el codigo generado en la aplicación tras escanear el codigo QR. 

- De clic en el boton **Validar codigo** para completar el proceso.  

<Note type="error"> Si deseas cambiar el método de autenticación, puedes hacer clic en la opción “Elegir otra manera de autenticar” antes de confirmar el método seleccionado. </Note>

Una vez la verificación del doble factor de autenticación sea confirmado con éxito se debe mostrar en pantalla los códigos de recuperación que se pueden usar en caso de que no pueda realizar con éxito el doble factor de autenticación usando las aplicaciones de autenticación.  

Se cuenta con el boton **Descargar**, el cual permite la descarga de los códigos, se recomienda guardarlos en un lugar seguro para que se puedan usar de ser necesario.   

<CenteredContainer>
![codigosderecuperacion](/payments/newpanel/migration/newuser/codigosderecuperacion.webp)
</CenteredContainer>

Al completar la autorización se debe abrir el **Dash** correspondiente al usuario creado, mostrando el Dashboard inicial.  

![homenewuser](/payments/newpanel/migration/newuser/dashes.webp)

## Paso 3 Inicio de sesión al Dash

Para acceder al Dash ingrese al enlace correspondiente a través de un navegador, se debe mostrar en pantalla los campos:
- Correo electrónico 
- Contraseña 

![loginnewpanel](/payments/newpanel/migration/newuser/loginhomees.webp)

Una vez completados los campos, de clic en el botón **Iniciar sesión** para acceder al sistema.  

Se debe mostrar en pantalla un mensaje solicitando el código de autenticación (Código de autenticación del 2FA) para poder acceder al sistema, seguido se debe contar con el campo para el ingreso de este.  

Se cuenta con la opción de marcar este dispositivo como uno de confianza, esto para que no se siga solicitando la autenticación al intentar acceder nuevamente desde el mismo dispositivo.  

Una vez ingrese el código otorgado en la aplicación de autenticación, de clic en el botón **Autenticar**. 

![login2fanewuser](/payments/newpanel/migration/newuser/login2fanewuser.webp)

Al completar la autenticación exitosa, el sistema permite acceder al **Dash**. 

![homenewuser](/payments/newpanel/migration/newuser/dashes.webp)

## Recuperar contraseña

<Note type="error"> Recomendamos verificar que las credenciales sean correctas e inténtarlo nuevamente, antes de restablecer la contraseña. </Note>

En caso de olvidar la contraseña, puede utilizar la opción **Olvidaste tu contraseña** para restablecerla.

![loginnewpanel](/payments/newpanel/migration/newuser/loginhomees.webp)

Donde digitaremos el correo electrónico y de clic en la validación reCaptcha, de ser necesario cumpla con el reto para continuar, seguido de clic en el botón **Enviar link**. 

<CenteredContainer>
![recuperacionpassword](/payments/newpanel/migration/newuser/recuperacionpassword.webp)
</CenteredContainer>

La pagina nos mostrará una confirmación en la cual nos indica que dentro de 5 minutos recibiremos el correo para restablecer nuestra contraseña.

<CenteredContainer>

</CenteredContainer>
