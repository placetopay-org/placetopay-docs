## Ingresar al módulo {{ id: 'enter_module', tag: '', label: '' }}

Al crear un link de pago, puedes personalizar todos los detalles necesarios para facilitar que tus usuarios accedan al pago de tu servicio. Tendrás la posibilidad de especificar información clave como la referencia del pago, descripción, monto, moneda y opciones de envío, ya sea por correo electrónico, WhatsApp o mediante un código QR. 
Además, podrás definir el número de pagos permitidos a través de dicho link, adaptándolo a las necesidades de tu negocio de manera sencilla y eficiente.   {{ className: 'lead' }} 

Para realizar la creación de un link de pago se debe ingresar al siguiente enlace: https://sites.placetopay.com/login   {{ className: 'lead' }}

<Note> El usuario y contraseña solicitado en esta etapa son los mismos que ingresan en el panel administrativo, en caso de no contar con los accesos solicitarlo vía correo electrónico de acuerdo a los canales establecidos.  </Note>

## Creación de un Link de Pago {{ id: 'create_payment_link', tag: '', label: '' }}

Para crear un nuevo link de pago se debe dar clic en el icono **+** {{ className: 'lead' }}

<CenteredContainer>
    <ImageZoom
      src="/Create_link.webp" 
      alt="link create example image" 
      className="w-full h-auto shadow rounded-md my-0 transition-transform duration-300 ease-in-out cursor-pointer hover:scale-110 focus:scale-110" 
    />
</CenteredContainer>


A continuación, se desplegarán los siguientes campos generales de un link de pago, los cuales deberán ser completados de la siguiente manera: {{ className: 'lead' }}

**Vincular a:** En este campo, debes seleccionar si el link de pagos estará asociado a un sitio o un micrositio. Elegir siempre la opción **Sitio**. {{ className: 'lead' }}

<Note> En caso de tener alguna configuración especial, se deberá asociar a un micrositio. </Note> 

**Sitio:** Digite el nombre del sitio al que deseas vincular el link de pago. En este caso, solo se mostrarán los sitios asociados al comercio. {{ className: 'lead' }}

**Nombre:** Ingresa el nombre con el que el link de pago aparecerá en el historial de links creados. Este será también el nombre con el que se compartirá el link de pago. {{ className: 'lead' }}

**Número de pagos permitidos:** En este campo, puedes especificar la cantidad de pagos aprobados que se permitirán a través del link de pago. **Si lo dejas vacío, los pagos aprobados serán ilimitados hasta que se alcance la fecha de expiración.** {{ className: 'lead' }}

**Tiempo de expiración del proceso de pago:** Este campo define el tiempo límite, en **minutos**, para que el proceso de pago expire. El conteo inicia cuando el usuario hace clic en el link de pago y comienza el flujo transaccional en el Checkout. Se recomienda establecer un rango de **10 a 15 minutos** para asegurar una experiencia fluida. {{ className: 'lead' }}

**Fecha de expiración:** Este campo establece la fecha límite de vigencia del link de pago. Una vez alcanzada esta fecha, el link quedará inhabilitado y no podrá ser utilizado. {{ className: 'lead' }}

Luego de diligenciar todos los campos se debe presionar el botón siguiente. {{ className: 'lead' }}

**Referencia:** La referencia es el identificador único de la transacción y será el valor utilizado para reconocerla en la consola administrativa de Evertec Placetopay. Este campo admite valores alfanuméricos y se asigna específicamente al link de pago creado. {{ className: 'lead' }}

<Note> Este campo no permite caracteres especiales (solo guion medio) y permite un máximo de 32 caracteres </Note>

**Descripción:** Este campo permite ingresar una breve descripción de la compra. Se acepta un valor alfanumérico con un máximo de 250 caracteres. {{ className: 'lead' }}

**Moneda:** En este campo, se selecciona el tipo de moneda con el que se generará el link de pago, y estará disponible en el flujo de pago. **(Esto aplica si se tienen habilitadas múltiples monedas en el panel administrativo).** {{ className: 'lead' }}

**Monto:** En este campo, ingresa el monto total que se va a pagar a través de este link de pago.

**Correos electrónicos de los destinatarios:** En este campo, ingresa uno o varios correos electrónicos a los que se enviará el link de pago **(este campo es opcional)**. Si deseas enviar el link a varios destinatarios, simplemente presiona "Enter" después de ingresar el primer correo para habilitar la opción de añadir más direcciones. {{ className: 'lead' }}
