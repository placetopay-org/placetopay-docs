## Compartir {{ id: 'share', tag: '', label: '' }}

En este apartado vamos a visualizar las diferentes maneras de compartir nuestro **link de pago** creado.{{ className: 'lead' }}

### QR {{ id: 'qr', tag: '', label: '' }}

En la vista reciente del enlace de pago creado (ver imagen anterior), la primera opción para compartirlo es a través del apartado QR. Al hacer clic en este, podrá visualizar el código QR ya generado, el cual puede copiar y compartir fácilmente.{{ className: 'lead' }}

La segunda opción consiste en hacer clic en los tres puntos ubicados en la esquina **superior derecha** de la pantalla. Luego, seleccione la opción Compartir para enviar el enlace de pago de manera sencilla.{{ className: 'lead' }}

Como primera opcion se visualiza la **URL** del link de pago el cual podemos copiar y pegar{{ className: 'lead' }}

## Pagos relacionados {{ id: 'relationship-payments', tag: '', label: '' }}

Al finalizar todo el proceso, en la vista correspondiente al link de pagos creado, se generará automáticamente un registro detallado de la transacción en el apartado Pagos relacionados. En esta sección, se podrán visualizar todos los pagos asociados que hayan sido procesados a través de ese link de pagos específico.{{ className: 'lead' }}

<Note>Ten en cuenta que el pago relacionado muestra solo información básica. Para ver detalles completos de los pagos recibidos, se recomienda consultarlos desde el panel administrativo.</Note>

En caso de no visualizar algún registro esperado, es importante verificar que el link haya sido utilizado correctamente y que el flujo de pago se haya completado sin interrupciones.{{ className: 'lead' }}

## Buscar links de pago {{ id: 'search', tag: '', label: '' }}

Para buscar en el historial de Links de Pagos, dirígete al menú lateral izquierdo de la plataforma y selecciona la opción **Links de Pagos**. Luego, haz clic en **Consultar** para visualizar el historial basado en el filtro preestablecido, que corresponde a la **fecha de creación**.{{ className: 'lead' }}

Si deseas realizar una búsqueda más específica, puedes eliminar el filtro predeterminado y utilizar la opción **Agregar filtro**. Esto te permitirá configurar criterios personalizados según las opciones disponibles, mostrando únicamente la información que se ajuste a los parámetros seleccionados. De esta forma, tendrás un acceso más preciso y eficiente a los datos de interés.{{ className: 'lead' }}

<br/>
### Email 
Una vez compartido el enlace a través del correo, el cliente podrá visualizarlo de la siguiente forma, tal como se ilustra a continuación:{{ className: 'lead' }}

<CenteredContainer>
  <div className='w-full sm:w-1/2 lg:w-1/2 sm:ml-10 sm:mr-6'>

Mediante el contenido que llega al correo, se permite realizar transacciones de forma rápida. Se cuenta con un código QR para facilitar el proceso de pago y se puede acceder al pago mediante el enlace ubicado en la parte inferior. Además, se incluye un resumen de pago como el monto, la referencia y la fecha límite.{{ className: 'lead' }}

<Note>
    La URL incluida en el correo es de uso único para el correo enviado. Si se requiere compartir el enlace de pago con varias personas, recuerde compartirlo mediante las opciones mencionadas anteriormente.{{ className: 'lead' }}
</Note>

### Vista de pago 

Al abrir el enlace de pago, ya sea a través del correo electrónico, el código QR, una URL directa o un mensaje de WhatsApp, se mostrará la siguiente información: la fecha de expiración, el valor a pagar, el comercio al que se realizará el pago, la descripción del pago, la referencia y el estado del enlace de pago.{{ className: 'lead' }}
