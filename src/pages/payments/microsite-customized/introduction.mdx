# Micrositio Personalizado
<br/>

Este servicio permite la consulta y gestión de facturas de pago conectandose al sistema del comercio. 
En esta sección, se explica como se realiza la busqueda de facturas y el asentamiento de un pago en la plataforma al sistema del comercio.

<Note>Para interactuar con este servicio, es necesario utilizar un lenguaje de programación compatible con REST APIs, lo que permite la comunicación eficiente entre sistemas.</Note>

El método de integración admite las siguientes operaciones clave:

- Consulta de facturas mediante referencia, número de documento o referencia alterna.
- Registro de pagos con la actualización automática del estado de la factura una vez procesado el pago.
